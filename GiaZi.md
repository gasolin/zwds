```
類別 花甲(物件):
    """
    子丑, 寅卯, 辰巳 ... 類推
    """
    定義 __初始化__(我, 名稱, 天干, 地支, 五行局):
        我.名稱 = 名稱
        我.五行局 = 五行局
        我.天干 = 天干
        我.地支 = 地支
        我.陰陽 = 天干.陰陽

    定義 __字串__(我):
        返回 我.名稱
```
