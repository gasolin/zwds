# Introduction #

```
類別 地支(物件):
    """
    地支參數
    """
    定義 __初始化__(我, 名稱, 陰陽, 方向, 時間):
        我.名稱 = 名稱
        我.陰陽 = 陰陽
        我.方向 = 方向
        我.時間 = 時間

    定義 __字串__(我):
        返回 我.名稱
```
