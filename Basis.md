# 簡介 #

術數基礎
```
類別 太極(物件):
    """
    道生萬物
    """
    定義 __字串__(我):
        返回 我.名稱


類別 兩儀(太極):
    """
    陰、陽
    """
    定義 __初始化__(我, 名稱, 數字):
        我.名稱 = 名稱
        我.數字 = 數字
```

```
陰 = 兩儀(名稱 = "陰", 數字 = 0)
陽 = 兩儀(名稱 = "陽", 數字 = 1)
```
